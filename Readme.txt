GitHub API Test

Этот проект содержит автоматический тест для работы с GitHub API на Python. Тест создает, проверяет наличие и удаляет репозиторий на GitHub.

## Установка зависимостей

1. Убедитесь, что у вас установлен Python (3.6 и выше).
2. Клонируйте этот репозиторий на свой компьютер.
3. Перейдите в папку проекта:
4. Установите зависимости:
 pip install -r requirements.txt

## Настройка переменных окружения

В файл.env в корне проекта и добавьте следующие данные:

- GITHUB_USERNAME — ваше имя пользователя на GitHub.
- GITHUB_TOKEN — ваш персональный токен доступа (вы можете создать его в настройках GitHub).
- REPO_NAME — имя репозитория, который будет создан.

## Запуск теста

Запустите тест с помощью следующей команды: pytest test_git_api.py или через Pycharm
